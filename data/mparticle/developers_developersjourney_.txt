Developer's Guided Journey to mParticlemParticle is a CDP that collects real time event data, directly within your application.
You can use mParticle to identify your users so that data is attributed correctly and to make it easy to build a comprehensive history of your users’ engagement. You can create data plans that maintain data integrity and quality. After your data is organized, you can create connections with third party data tools.
Finally, you can use your data to improve your user engagement and retention with mParticle’s analytics and predictive tools, Indicative and Cortex.
Following is a chronological overview of each step developers must follow when implementing mParticle. Use the provided links to find the most relevant documentation for each stage. If you are unfamiliar with a particular topic, begin by reading any provided guides before progressing to the related API or SDK reference.
